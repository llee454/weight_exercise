Analysis Data Readme
====================

Exercise Data 
-------------

### Source

https://www.mapmyrun.com/workouts/dashboard/lifetime

### Data Format

1. Date Submitted,
2. Workout Date,
3. Activity Type,
4. Calories Burned (kCal),
5. Distance (mi),
6. Workout Time (seconds),
7. Avg Pace (min/mi),
8. Max Pace (min/mi),
9. Avg Speed (mi/h),
10. Max Speed (mi/h),
11. Avg Heart Rate,
12. Steps,
13. Notes,
14. Source,
15. Link
